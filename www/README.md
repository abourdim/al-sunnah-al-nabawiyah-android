# 📜 Prophetic Sunnah — السنة النبوية بين أهل الفقه وأهل الحديث

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **السنة النبوية بين أهل الفقه وأهل الحديث (The Prophetic Sunnah Between Jurists and Hadith Scholars)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on understanding hadith, fiqh, and their relationship
- 8 reflections on common misunderstandings of the Sunnah
- Daily habit tracker with streaks, self-assessment quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes (Teal, Night, Ocean) with localStorage persistence
- Narrator with karaoke, voice/speed/pitch controls, book mode
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
